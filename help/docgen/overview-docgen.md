# 文档生成API教程

![文档生成API横幅](../assets/docgenhero.jpg)

文档生成API可根据WordPDF和JSON数据创建Word和Word文档。

>[!NOTE]
>
>文档生成API包含在PDF服务API中。

## 创建模板

文档生成API接受文档模板（带有模板标签）以及输入数据来生成最终文档。 最后文档是通过基于与数据输入相对应的实际值，用动态内容替换文档模板中的所有模板标签来生成的。
